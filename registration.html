Abstracts Submission


The authors are asked to send a Word document or LaTeX file containing the abstract to j2ifam@ugr.es before January 15th, 2012. The e-mail should specify whether you wish to contribute an oral communication or/and a poster.
In order to prepare the abstracts, the document should be generated using one of the attached template files, in the Word or LaTeX format. The template files can be downloaded from this web page. We kindly request that the authors follow the provided templates.

Posters

If you are going to present a poster, please consider that the size of the poster holder is 90 cm (width) x 120 cm (height).
