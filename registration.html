Registration and Abstract submission


Please click here to register.
The authors are asked to send a Word document or LaTeX file containing the abstract to jomiste@ucm.es before August 15th, 2021. The e-mail should specify whether you wish to contribute an oral communication or/and a poster.
In order to prepare the abstracts, the document should be generated using one of the attached template files, in the Word or LaTeX format. The template files can be downloaded from this web page. We kindly request that the authors follow the provided templates.
